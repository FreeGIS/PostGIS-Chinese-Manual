PostGIS-Chinese-Manual

PostGIS中文手册，富含大量实际案例和使用场景。

一 安装环境：

1.1 安装nodejs

1.2 全局安装gitbook

npm install gitbook-cli -g

1.3 获取仓库并安装启动插件

git clone git@github.com:FreeGIS/PostGIS-Chinese-Manual.git
cd PostGIS-Chinese-Manual
gitbook install

1.4 启动项目

gitbook serve

页面预览地址：http://localhost:4000
